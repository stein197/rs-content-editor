# ROCKSTONE Content Editor

## NPM скрипты
- `webpack` Сборка проекта
- `webpack:dev` Сборка проекта в режиме разработки. В режиме разработчика происходит следующее - React собирается в режиме разработчика, генерируются source maps, отключается минимизация, компилятор TypeScript менее строгий

### Переменные webpack
Использование:
```
npx webpack --env <...переменные>
```
где `<...переменные>` одно из:
- `dev` Собирает проект в режиме разработчика
